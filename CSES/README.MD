This directory contains my solutions to the CSES Problem Set.
-
**Folder Structure**
-
The repository is organized into subfolders, each one representing a category from the official CSES problem set:

    CSES
    │
    ├── Introductory_Problems/
    │   ├── weird_algorithm.cpp
    │   ├── missing_number.cpp
    │   └── ...
    │
    ├── Sorting_and_Searching/
    │   ├── concert_tickets.cpp
    │   ├── restaurant_customers.cpp
    │   └── ...
    │
    ├── Dynamic_Programming/
    │   ├── dice_combinations.cpp
    │   ├── minimizing_coins.cpp
    │   └── ...
    │
    ├── Graph_Algorithms/
    │   ├── counting_rooms.cpp
    │   ├── labyrinth.cpp
    │   └── ...
    │
    ├── Range_Queries/
    ├── Tree_Algorithms/
    ├── Mathematics/
    ├── String_Algorithms/
    └── Additional_Problems/

**File Naming Convention**

Each solution file is named after the original problem title using snake_case.
Examples:

    weird_algorithm.cpp

    increasing_array.cpp   


**Problem Links**
-

Every solution file begins with a header containing the link to the problem, using the following template:

    https://cses.fi/problemset/task/<id>/

Example inside a .cpp file:

    // Problem: Weird Algorithm
    // Link: https://cses.fi/problemset/task/1068/

**Notes**
-
- All solutions are written in C++ (C++20).
- Each subfolder groups problems by topic to make studying and reviewing easier.

- This collection helps improve essential competitive programming skills, such as:
  - Algorithmic thinking
  - Dynamic Programming
  - Graph Theory
  - ...
