# Data Structures UTP - HackerRank

This folder contains my solutions to problems from the **HackerRank Data Structures UTP contest**.

## Structure

Each solution file corresponds to a specific problem from the contest.

At the **top of every file**, I include a header with the **link to the original problem**, so it can be easily accessed and reviewed.

## Problem Links

All problems follow this structure:

https://www.hackerrank.com/contests/data-structure-utp/challenges/ problem-name

### Example
https://www.hackerrank.com/contests/data-structure-utp/challenges/josephuss-problem

## Notes

- Solutions are mainly written in **C++**.
- Each file includes a reference link to the original problem.
- The goal is to practice data structures and problem solving.
