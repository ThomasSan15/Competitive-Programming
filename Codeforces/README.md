# Codeforces Solutions

This folder contains my solutions to problems from **Codeforces**.

## Structure

The `Codeforces` directory is organized into **subfolders**, where each subfolder corresponds to a specific **Gym contest** or **Contest** from which the problems were selected.

Each Gym folder contains the solutions to the problems solved from that contest.

Example structure:


    Codeforces/
    │
    ├── Contest_XXXX/
    │ ├── A.cpp
    │ ├── B.cpp
    │ └── C.cpp
    │
    ├── Gym_XXXX/
    │ ├── A.cpp
    │ └── D.cpp
    │
    ├── Gym_XXXX/
    │ ├── A.cpp
    │ ├── B.cpp
    │ └── E.cpp
    ├── Problemset´s/
    │ ├── ASCII Art Contest.cpp
    │ ├── Ashmal.cpp
    │ ├── Carnival Wheel.cpp
    │ └── ...

    



## Notes

- Each file corresponds to a **problem solved during practice or contests**.
- Solutions are mainly written in **C++**.
